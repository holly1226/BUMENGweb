# 📖 捕梦 — 游戏说明

"捕梦"是一款 AI 主持的多人文字角色扮演游戏（TRPG）。你和朋友们各自扮演不同角色，在 AI 主持人（DM）的引导下，探索由 AI 实时生成的奇幻世界，共同书写冒险故事。

---

## 一、基本流程

### 1. 创建或加入房间

1. 打开游戏页面，输入你的**玩家名**
2. 点击**「创建房间」**→ 系统生成 6 位房号
3. 把房号分享给朋友，朋友点击**「加入房间」**输入房号
4. 房间列表会显示所有在线玩家，房主有 👑 标识

### 2. 等待大厅

进入房间后进入等待大厅（LOBBY）：

- **世界观偏好** — 输入你想要的风格（如：中世纪奇幻、赛博朋克、武侠江湖）
- **角色意愿** — 输入你想扮演的角色类型（如：流浪剑客、神秘法师、赏金猎人）
- 点击**「提交意向」** 报名

> 💡 提交偏好后，其他玩家也能看到你的偏好

### 3. 开始游戏

- **只有房主（👑）能点击「开始游戏」**
- 非房主玩家看到「👑 等待房主 XXX 开始...」
- 点击开始后，AI 根据所有玩家的偏好生成剧本和角色卡
- 生成期间会有**大逃sha类场景背景**和**开场剧情**

> ⚠️ 等人齐了再开始！开始后无法再加入新玩家。

### 4. 自我介绍阶段

游戏开始后，每位玩家进入**全屏自我介绍界面**：

- 查看 AI 为你生成的专属角色卡（身份、背景、技能）
- 输入一段**角色自我介绍**（角色叫什么、怎么来到这里的）
- 所有人完成后，游戏正式开始

### 5. 游戏中

- **自由发言**：在对话框输入行动描述，AI 实时回应
- **公屏聊天**：所有玩家互相可见
- **DM 私密消息**：AI 可能单独对你说话
- **场景切换**：AI 会根据剧情推进自动更换场景背景
- **回合推进**：AI 自动判断推进时机

## 二、房主机制

| 操作 | 说明 |
|------|------|
| 创建房间 | 自动成为房主 👑 |
| 开始游戏 | 只有房主能点击 |
| 断开连接 | 房主离开后，自动转移给房间内下一位玩家 |
| 所有人离开 | 房间保留，新玩家可加入（新加入的第一人自动成为房主） |

## 三、如何写出好的行动

### ✅ 好的描述

- "我小心翼翼地推开吱呀作响的木门，探头望向黑暗的走廊深处"
- "我拔出长剑横在胸前，对守卫冷静地说：我们只是路过，不想惹麻烦"
- "我蹲下检查地上的血迹，用指尖蘸了一点，放到鼻尖闻了闻"

> 公式：**动作 + 意图 + 细节**，给 AI 足够的发挥空间

### ❌ 不太好的描述

- "我走过去" — 太简短，AI 难以展开
- "我一刀砍死 BOSS" — 跳过故事，破坏体验
- "我什么也不做" — 浪费一次行动机会

## 四、角色创建建议

开局可自定义角色偏好，AI 会据此生成角色卡：

- **世界观风格**：奇幻、修仙、科幻、克苏鲁、赛博朋克、中世纪...
- **角色定位**：近战战士、远程法师、敏捷游侠、治疗牧师、诡计盗贼...
- **故事基调**：轻松冒险、紧张恐怖、史诗战争、侦探推理...

## 五、界面说明

| 区域 | 功能 |
|------|------|
| **左侧面板** | 在线玩家列表（👑标识房主）、房间信息、回合状态 |
| **中央** | AI 生成的场景背景图 + 对话历史 |
| **右上面板** | 角色卡、背包/物品栏 |
| **底部输入区** | 文本输入框，输入行动后发送 |

## 六、FAQ

**Q: 可以几个人一起玩？**
A: 推荐 2-5 人，体验最佳。理论上无上限。

**Q: 需要大家同时在线吗？**
A: 是的，游戏是连贯推进的，需要玩家同时在线轮流行动。

**Q: 支持存档吗？**
A: 暂不支持。建议在结局时截图保存精彩时刻。

**Q: 手机能玩吗？**
A: 已适配移动端布局，浏览器直接打开即可。

**Q: AI 为什么给我和朋友生成了一样的角色？**
A: 已修复。系统会强制每个玩家获得不同的角色身份和背景。

**Q: 怎么切回大厅？**
A: 左侧面板底部有「🏠 返回大厅」按钮（游戏结束后可点击）。
